# AlmaSampler MVP

Browser-first music sampling MVP inspired by Serato Sample and a lightweight piano-roll workflow.

## Milestone 1

- pnpm workspace monorepo
- Vue 3 web app scaffold
- Fastify API scaffold
- shared domain package
- browser audio upload and decode
- duration validation
- basic waveform rendering shell

## Workspace Layout

```text
apps/
  api/
  web/
packages/
  shared/
```
